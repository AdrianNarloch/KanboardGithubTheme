## Description

This plugin provides the **Github** theme for Kanboard, featuring:

- Modern, clean, and accessible color palette
- GitHub-inspired design system with Primer colors
- Comprehensive styling for all Kanboard components
- Professional and minimalist appearance
- Full responsiveness and mobile support

## Features

- **Primary Color**: GitHub Blue (#0969da)
- **Clean Typography**: System fonts with optimized rendering
- **Accessible Design**: WCAG compliant color contrasts
- **Modern Components**: Updated styling for buttons, forms, boards, and cards
- **Responsive Layout**: Works seamlessly on all screen sizes

## Installation

1. Download the zip file of this repository
2. Extract the `KanboardGithubTheme` folder into your Kanboard `plugins/` directory
3. The theme will be automatically applied
